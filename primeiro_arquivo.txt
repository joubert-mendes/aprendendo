Git é o sistema mais usado no mundo ! um projeto de codigo aberto ! onde tem varias funçoes para simplificar a vida do desenvolvedor onde seus principais comandos são:

.Comando para entrar com login do github

Git config --global user.name 
config --global user.email

.Comando para criar um novo projeto 
Git init

.Comando para copiar repositorio do git via link
Git clone

.Comando que adicona arquivos ao repositorio
Git add nome do arquivo
ou Git add . para adicionar todos

.Comando para fazer commit dos arquivos adicionados 
Git commit 
Git commit -m para "deixar mensagem"

.Comando para trazer conteudo do repositorio remoto
Git pull

.Comando para enviar arquivos alterados no computador local para o repositorio
Git push 

.comando para ver arquivos modificados 
Git status

.Comando para criar uma tag de informaçao para versoes
Git tag
